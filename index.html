What is Robotic Process Automation

RPA or Robotic process Automation, refers to a piece of software (a robot) configured to perform mundane tasks the way a human would with the traditional mouse and keyboard.

⥥ Following is a simple example of how RPA works


01. Robot can monitor a mailbox and read email from specific user

02. It downloads attached excel file

03. Robot then Logs in to your accounting system using a Username and Password

04. Posts the contents of excel row by row to the accounting system, maintain log of each entry with a Date/time stamp

05. Upon Completion, Emails the completion status to the stakeholders

What are the components of a RPA platform?

Typical Robotic Process Automation platform would have the following components ,When used together they form a very powerful ecosystem for process automation

Workflow Designer

Also known as Studio, helps configure the robot used by RPA implementation partners and RPA developers​​

Recorder

Recorder provides the ability to capture user actions without the need to program anything

Control Panel

Software that allows powerful controls mechanism for Robots remotely through a web interface

Process Runner / Bot

A programmed Robots are the output from the Development Platform which is then deployed to the business environment
